Fungsi Kuadrat DP

Ini cuma rangkuman aja. Kalo lo ada yang gak ngerti kenapa begini, liat lagi teorinya.


Bentuk umum fungsi kuadrat

$$ F(x) = ax^{2} + bx + c, \quad a \neq 0 $$




  1. Jika koefisien a nilainya positif maka kurva terbuka ke atas, jika negatif terbuka ke bawah.
  2. Koefisien c menyatakan letak si kurva relatif terhadap sumbu y.
  3. Jika a . b > 0 maka titik puncaknya di kiri sumbu y. Jika a . b < 0 maka titik puncaknya di kanan sumbu y.

Titik puncak/simetri

sumbu:

$$ x_{p} = \frac{-b}{2a} \qquad y_{p} = \frac{-D}{4a} \qquad y_{p} = F (x_{p}) $$





Titik puncak menggunakan turunan

$$ f(x) = ax^{2} + bx + c \\ f'(x) = 2ax +b \\ f'(x) = 0 \\ 2ax + b = 0 \\ x = \frac{-b}{2a} \\$$










Dan nilai x itu lo masukin deh ke fungsi awal, dapetlah nilai y puncaknya.


Translasi fungsi kuadrat

$$ y = ax^{2} \ \rightarrow \ \binom{xp}{yp} \Rightarrow \ y-yp = a \ (x-xp)^{2} $$





Fungsi Kuadrat dengan akar-akar

$$ F(x) = a (x-x_{1})(x-x_{2}) $$
